# Frontend

This folder contains the frontend application for the Job Recommendation System (Vite + React + TypeScript + Tailwind + shadcn-ui).

## Running locally

Install dependencies and start the dev server:

```bash
cd frontend
npm install
npm run dev
```

## Editing

Edit files in your preferred IDE, commit, and push. The app is designed to work locally without external cloud tooling.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Notes

Removed references to third-party cloud editing services. If you need help running or deploying the frontend, open an issue or ask for guidance.
